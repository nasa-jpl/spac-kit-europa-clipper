SPaC-Kit Europa Clipper Plugin
==============================

A subset of the CCSDS packets used by the `Europa-Clipper mission <https://science.nasa.gov/mission/europa-clipper/>`_ are described using CCSDSpy packet definitions in this repository.
The science packets for the `ECM <https://science.nasa.gov/mission/europa-clipper/spacecraft-instruments-ecm/>`_, `MISE <https://science.nasa.gov/mission/europa-clipper/spacecraft-instruments-mise/>`_ and `SUDA <https://science.nasa.gov/mission/europa-clipper/spacecraft-instruments-suda/>`_ instruments are documented in particular.

The source code describing the packets can be found in our `github repository <https://github.com/nasa-jpl/spac-kit-europa-clipper>`_.

The `SPaC-kit <https://github.com/CCSDSPy/spac-kit>`_ documentation generator is used to produce the human readable documentation of the packets visible below.
